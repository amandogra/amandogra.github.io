Create an account at https://openshift.redhat.com

Install Yeoman by following the steps mentioned in Yeoman Setup Guide

Install Angular Fullstack Yeoman generator
code https://github.com/DaftMonk/generator-angular-fullstack

Scaffold your app
yo angular-fullstack krishnatv

Deploy it on Openshift
yo angular-fullstack:openshift
